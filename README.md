# Budget Application - GitOps Deployment with FluxCD

## Overview
This repository demonstrates a GitOps-based continuous deployment (CD) setup using FluxCD and Helm for deploying applications to Kubernetes.

## Tech Stack
- Kubernetes
- FluxCD
- Helm
- Prometheus

## Features
- GitOps-based deployment using FluxCD
- Helm charts for Kubernetes resource management
- Automated application deployment on Git changes
- Monitoring setup using Prometheus

## Deployment Workflow
1. Code/configuration changes pushed to Git
2. FluxCD detects changes
3. Helm chart is applied to Kubernetes cluster
4. Application is deployed/updated automatically

## Purpose
This project was built to understand GitOps workflows, Kubernetes deployments, and monitoring setup in a DevOps environment.
